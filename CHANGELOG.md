# Changelog

All notable changes to SocialX are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-03-23

### Improved
- Refreshed the README to better explain architecture, product flow, and interview talking points.
- Added visual diagrams for the navigation map and system architecture.
- Cleaned repository metadata to better match the public GitHub project.
- Removed generated Expo and dependency artifacts from version control.

## [1.0.0] - 2024-12-21

### Added
- Initial release of SocialX.
- Mock email/password authentication and simulated Google sign-in.
- Interactive social feed with likes, comments, and share actions.
- Image discovery gallery with modal previews.
- User profiles with basic statistics and post summaries.
- Dark and light theme toggle with motion-driven transitions.
- Drawer and bottom-tab navigation for core app sections.
- Settings, help, privacy, and about screens for a complete demo flow.
